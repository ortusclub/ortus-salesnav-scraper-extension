          Tune for your machine

          Slow mode lives at the top of every screen — these are the rest. Defaults are sensible.

          Corner window
          Shrinks the scraping tab into the top-right corner so you can keep working. On by default — keeps the tab visible (and not throttled).
          Auto-open status window
          When a scrape starts, automatically open the floating status strip. It keeps the scrape ticking faster — minimize it, don't close it.
          Prevent Mac sleep
          Run this in Terminal during long batches so macOS doesn't sleep mid-scrape. ⌃C to stop.
          Chrome Memory Saver
          Auto-handled — every scrape tab is pinned as non-discardable so Chrome can't freeze it mid-run. If you still see stalls, open Chrome's performance page and confirm the LinkedIn tab isn't dimmed.
