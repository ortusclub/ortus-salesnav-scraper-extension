# Sketch Manifest

## Design Direction

Refresh the Sales Nav scraper popup from generic dark-utility (`#0f1117`, blue/green chips, DM Sans) to the Ortus **club-editorial** language already used by the LinkedIn DM Assistant: warm cream paper (`#FDFBF7`), ink + gold palette, Newsreader serif for display + Geist sans for body + Geist Mono for labels and eyebrows. Rounded 16px popup with newsprint noise overlay. Information architecture is open — three layouts to compare.

This is also the moment to stop reporting sub-25 anonymized pages as "partial" — those leads were never visible to the searcher's account in the first place, so the UI shouldn't read like something went wrong.
